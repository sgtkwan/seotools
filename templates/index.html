File Structure Requirements

How to Structure Your File:

Your Excel/CSV file must follow this exact column structure for the AI to properly classify your keywords.

Example File Structure:
Keywords Brand Category 1 (ie Branded/Generic) Category 2 Category 3 ... Category X
keyword1 Brand (ie groupm) tag1 (ie brand) tag1 tag1 ... tag1
keyword2 tag2 (ie generic) tag2 tag2 ... tag2
keyword3 tag3 tag3 ... tag3
keyword4 ... tag4
keyword5 ...
Instruction-only option: You may leave a category column blank (no tags under the header) to let the AI infer suitable tags. You'll be prompted to add optional instructions for such columns on the next page.
Required Columns
  • Column 1: Keywords
    The search terms you want classified (one per row)
  • Column 2: Brand
    Brand name so the AI can classify what is branded
  • Column 3+: Classification Columns
    Each column is a category. Either list tags under the header (predefined) or leave cells blank to let the AI infer tags (you can add instructions on the next page).
Important Rules
  • Minimum 3 columns (Keywords + Brand + 1 Category)
  • Header names stay the same in the output file
  • Two modes for categories: - Predefined tags: list tags under the header - Instruction-only: leave cells blank; you'll add instructions on the next page
  • One tag per column; blanks are allowed if none apply
Technical Requirements:
  • Supported formats: CSV, XLS, XLSX
  • Maximum file size: 16MB
  • No unrelated empty columns. Category columns may be intentionally left blank to let the AI infer tags.
  • First row must be headers
  • Data starts from row 2
